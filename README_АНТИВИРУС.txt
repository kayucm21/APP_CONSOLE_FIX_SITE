Смотрите файл: ИНСТРУКЦИЯ_ПО_СБОРКЕ.md


🔧 ПАРАМЕТРЫ СБОРКИ:

Полная сборка (рекомендуется):
   .\BUILD_ALL.ps1

Без подписи:
   .\BUILD_ALL.ps1 -SkipSign

Без упаковки UPX:
   .\BUILD_ALL.ps1 -SkipPack

Только компиляция:
   .\BUILD_ALL.ps1 -SkipSign -SkipPack


💡 СОВЕТЫ:

1. Если BUILD_ALL.ps1 не запускается:
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process

2. Если антивирус блокирует:
   • Добавьте папку в исключения
   • Временно отключите антивирус
   • Используйте базовую сборку без доп. защит

3. Если EXE не запускается:
   • Запустите от администратора
   • Разрешите в Windows SmartScreen
   • Добавьте в исключения Defender
